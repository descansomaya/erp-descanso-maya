Configuración
Apps Script → Implementar → Nueva implementación → Aplicación web → Ejecutar como: Yo → Acceso: Cualquier usuario → Implementar. Copia la URL /exec.

⚠️ Columnas requeridas en el Sheets:
· Hoja productos: agregar material_hilo_id y stock_disponible
· Hoja artesanos: agregar esquema y tarifa
· Hoja materiales: verificar que tenga tipo
· Hoja abonos_clientes: id · pedido_id · cliente_id · monto · nota · fecha
· Hoja receta_pedido: id · pedido_id · material_id · cantidad · color · notas · fecha_creacion
· Hoja producto_variantes: id · producto_id · talla · tipo_hilo · terminado · precio_venta · costo_produccion · tubos_hilo · material_hilo_id · activo
· Hoja pedido_detalle: agregar columnas variante_id y variante_label
· Hoja ordenes_produccion: verificar que tenga columna artesano_brazos_id
En Google Drive: clic derecho en la imagen → Compartir → Cualquiera con el enlace → copia el ID del URL y pon: https://drive.google.com/uc?id=TU_ID
